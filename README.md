# [mwid.dev](https://mwid.dev/)

My personal portfolio and digital resume.

Built with [React](https://react.dev/), [Next](https://nextjs.org/) and [Sass](https://sass-lang.com/).

## Development

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm run dev
```

Build static assets:

```bash
pnpm run build
```
